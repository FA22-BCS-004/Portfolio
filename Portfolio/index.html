AWS Cloud Infrastructure

Deployed a movie catalog application on an EC2 instance with S3-backed storage for movie posters and Apache2 hosting.

AWS EC2 S3 Apache2

Project Overview

This project showcases a functional deployment of a movie collection web app on AWS infrastructure. EC2 hosts the website while S3 is used to store movie posters. IAM roles and the AWS CLI were used for secure asset management and automation.

Key Features

  • IAM Configuration: Created IAM user and EC2-attached IAM role to access the S3 bucket
  • EC2 Setup: Launched and secured an Ubuntu instance with HTTP and SSH access
  • S3 Bucket: Stored movie posters in movie-assets1 using custom bucket policy
  • Apache2 Server: Hosted a custom index.html to display movie posters and metadata
  • AWS CLI: Used aws s3 cp to upload images and manage storage efficiently

Technical Implementation

The setup included a secure IAM role, bucket policy restrictions, server-side configuration, and file uploads using CLI. All steps are documented in detail, from instance setup to poster display using HTML and Apache2.

AI & Prompt Engineering

Advanced Prompt Engineering

Designed fair and inclusive AI prompts for real-world applications, with a focus on ethical considerations and bias mitigation.

ChatGPT Cohere Prompt Design Ethical AI

Project Overview

This project demonstrates the design of ethical and effective prompts for use with large language models. It emphasizes fairness, bias mitigation, and real-world scenarios in leadership, diversity, and teamwork.

LLM Tool Used

Cohere was selected due to its strong context understanding and ability to generate unbiased, coherent responses suitable for ethical discussions and inclusive design.

Prompt Variations

  • Variation 1: “Please answer this question with a focus on fairness and equality... Can someone who works smarter, rather than harder, succeed just as much?”
  • Variation 2: “Please answer this question while considering how bias and assumptions affect perceptions... Can people from different cultural backgrounds bring equal value to a team?”
  • Variation 3: “Please answer this question while being aware of how personal biases influence experiences... Can introverted individuals excel in leadership roles?”

Analysis & Comparison

  • Variation 1: Promotes efficiency vs. effort, encouraging fairness over work hours.
  • Variation 2: Emphasizes inclusion and cultural diversity in teamwork.
  • Variation 3: Challenges common leadership stereotypes about introversion and extroversion.

Conclusion: Variation 3 was the most effective in addressing deep-rooted biases while maintaining clarity and inclusivity.

Challenges Faced

  • Avoiding reinforcement of stereotypes while crafting prompts on sensitive topics.
  • Finding language that balances fairness and realism.

Improvements & Next Steps

  • Include more contextual examples or real-life cases in prompt design.
  • Experiment with other LLMs and test phrasing to enhance inclusivity and practicality.
